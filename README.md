





# Хатинка Лапок
Про проєкт

Хатинка Лапок — це веб-додаток для пошуку та знайомства з тваринами, які потребують дому. Мета проєкту — надати користувачам можливість переглянути доступних тварин, дізнатись про їхні особливості та залишити заявку на знайомство для потенційного всиновлення.

Проєкт включає інтерактивні секції: список тварин з фільтрацією, FAQ, слайдери успішних історій та модальні вікна для детальної інформації та оформлення заявки.

# Макет

Figma макет проєкту

API / Бекенд

Документація API

 # Використовуються наступні маршрути:

/api/animals — отримання списку тварин

/api/categories — отримання списку категорій

/feedbacks — отримання відгуків користувачів

/orders — відправка заявки на знайомство з твариною
